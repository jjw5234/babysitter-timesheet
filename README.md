##Getting Started

### How to Run the Server

1. Redis
   Download redis-cli

```
yarn redis-cli
```

2. Run Docker
   Navigate to the server folder

```
docker-compose up -d
```

3. Start the redis-server

```
redis-server
```

4. Start the server

```
yarn start
```

### How to Run the Client

1. Navigate to the client folder

```
yarn start
```
